## ✨ Features

- 🪨 Rock, 📄 Paper, ✂️ Scissors gameplay
- 🤖 Random AI opponent
- 📊 Real-time score tracking
- 🎉 Win and 😢 Lose screens
- 🔊 Sound effects
- 🔄 Reset game
- 📱 Responsive design

---

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS
- JavaScript (ES6)

---

## 🎮 How to Play

1. Choose **Rock**, **Paper**, or **Scissors**.
2. The computer makes a random choice.
3. The winner is determined automatically.
4. Scores update after each round.
5. The first player to reach **3 points** wins the game.
